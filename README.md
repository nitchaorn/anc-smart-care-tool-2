
# โปรแกรม ANC Smartcare Tool โรงพยาบาลแปลงยาว

โปรแกรมช่วยคำนวณอายุครรภ์ ประเมินการดูแลหญิงตั้งครรภ์ การได้รับวัคซีน การติดตามนัดต่อเนื่องตามแนวทางฝากครรภ์คุณภาพ สำหรับใช้งานภายใน โรงพยาบาลแปลงยาว จังหวัดฉะเชิงเทรา

## 👩‍⚕️ ผู้พัฒนา
- **ชื่อ:** นิชนารถ ปานอ่ำ
- **ตำแหน่ง:** พยาบาลวิชาชีพชำนาญการ
- **หน่วยงาน:**คลินิคฝากครรภ์ โรงพยาบาลแปลงยาว จังหวัดฉะเชิงเทรา
- **วันที่พัฒนา:** เมษายน 2569

## 📌 วัตถุประสงค์
พัฒนาเพื่อใช้งานภายในโรงพยาบาลแปลงยาว สำหรับประกอบการประเมินผู้ป่วย และใช้เป็นเครื่องมือในการทำ R2R / CQI

## ⚠️ สิทธิ์การใช้งาน
โปรแกรมนี้พัฒนาโดย นิชนารถ ปานอ่ำ
หากนำไปใช้หรืออ้างอิง กรุณาระบุแหล่งที่มาด้วยค่ะ
